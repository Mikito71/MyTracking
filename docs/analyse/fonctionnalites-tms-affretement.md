# Fonctionnalites TMS et affretement

## Objectif

Centraliser les fonctionnalites qu'un TMS moderne et un logiciel d'affretement peuvent proposer, afin d'alimenter le cadrage fonctionnel de MyTracking.

Cette liste sert de reference de travail. Elle n'est pas un engagement de MVP complet.

## Ordre MVP propose pour MyTracking

1. Admin plateforme : societes, bases, acces.
2. Societe, agences, personnel et profils.
3. Clients et affretes.
4. Expeditions simples.
5. Affretement : dossier, affrete, prix achat/vente, marge, confirmation.
6. Exploitation : planning, tournee, statuts.
7. Facturation.
8. Mobile chauffeur.
9. Import ODBC WinDev/HFSQL.
10. Reporting.
